# 📊 Messy E-Commerce Sales Data Analysis Using Excel

---

## 📝 Project Overview

This project focuses on cleaning, analyzing, and visualizing a messy e-commerce sales dataset using Microsoft Excel/WPS Spreadsheet.

The raw dataset contained several real-world data issues such as:

- Missing values
- Duplicate records
- Invalid dates
- Incorrect totals
- Negative values
- Inconsistent categories
- Wrong formatting
- Extra spaces

The objective of this project was to clean the dataset, perform business analysis, and create an interactive dashboard for decision-making.

## 🔍 Business Insights

Some key business insights identified from the analysis include:

- Sports category generated the highest sales.
- February 2025 recorded peak revenue.
- Cash on Delivery was the most frequently used payment method.
- Returned orders were significantly high.
- Electronics products contributed strongly to revenue generation.
- Monthly sales fluctuated significantly across the year.
